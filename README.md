# 🚀 Abdul Mateen — Portfolio Website

A personal portfolio website built with vanilla HTML, CSS, and JavaScript.

🌐 **Live Site → [abdulmateen.tech](https://abdulmateen.tech)**

---

## ✨ Features

- 🎨 Deep space aesthetic with neon mint & electric violet palette
- 🕸️ Interactive neural network canvas animation in the hero
- ✍️ Typewriter role animation
- 🖱️ Custom cursor with smooth follow effect
- 📱 Fully responsive — mobile, tablet, and desktop
- 🔍 Scroll-reveal animations throughout
- ⚡ Zero frameworks — pure HTML, CSS, and JS

---

## 🗂️ Sections

- **Hero** — Name, animated role, and intro
- **About** — Background, stats, and focus areas
- **Projects** — Featured AI/ML work with metrics
- **Skills** — Technical stack overview
- **Journey** — Education, experience, and certifications
- **Contact** — Links and email

---

## 🛠️ Built With

- HTML5
- CSS3 (custom properties, grid, flexbox, animations)
- Vanilla JavaScript (Canvas API, IntersectionObserver)
- Google Fonts — Plus Jakarta Sans, Inter, JetBrains Mono
