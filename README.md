## Projects

### 🏨 SmartStay AI — Hotel Front Desk Chatbot
FastAPI-based AI assistant using Qwen LLM with RAG and CRM features. Includes a full real-time voice pipeline with Moonshine ASR + Kokoro TTS. Connected to Calendar, Calculator, and OpenWeather APIs for live hotel operations. Retrieval faithfulness improved from ~0.65 to ~0.81 using RAGAS evaluation and reranking.

**Stack:** FastAPI · Qwen LLM · RAG · Moonshine ASR · Kokoro TTS · Python

---

### ⚡ TranslateSub AI — Real-Time Subtitle Translator
Chrome extension for real-time subtitle generation and multilingual translation of streaming content. Low-latency pipeline using Faster-Whisper, WebSockets, and sliding-window audio segmentation. End-to-end latency brought down to ~600–700ms.

**Stack:** Faster-Whisper · FastAPI · WebSockets · Chrome Extension API

---

### 🔥 EVEv2 VLM CUDA Optimization
Multi-stream parallelism on EVEv2.0's dual-branch FFN. Achieved a **1.26× end-to-end speedup** with a 4.5hr evaluation time reduction — zero retraining, zero additional GPU memory overhead. Profiled using CUDA events.

**Stack:** CUDA · PyTorch · EVEv2

---

### 🛰️ Region-Phrase Alignment for RS-VLMs
Extended DGTRS-CLIP with dual-encoder contrastive learning for remote sensing Vision-Language Models. Benchmarked with Recall@K across I2T and T2I directions. Achieved a **+9% I2T R@1 on RSITMD** and +3.2% mean Recall overall.

**Stack:** CLIP · PyTorch · Contrastive Learning

---

### ♿ VoiceLens — Accessibility OCR Tool
Mobile app combining OCR text detection and TTS to read signs, menus, and printed documents aloud in real-world conditions. Built to be a practical accessibility tool for low-vision users.

**Stack:** OCR · TTS · Mobile
